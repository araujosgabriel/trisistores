# Trisistores

O **Trisistores** é um kit educacional concebido para democratizar o acesso ao ensino prático de eletrônica e robótica nas escolas. A proposta é simples: oferecer protótipos didáticos de baixo custo, escaláveis e modulares, organizados em três níveis de complexidade — do primeiro contato com corrente elétrica até aplicações programáveis com microcontrolador. Cada nível contém atividades práticas que exploram conceitos fundamentais (polaridade, circuito fechado, leitura de sensores, temporização, e automação), sempre usando metodologias ativas de ensino.


O sistema é dividido em três níveis:

| Protótipo | Objetivo |
|--------|--------|
| Trisistor P | Introdução à eletrônica |
| Trisistor M | Circuitos eletrônicos |
| Trisistor G | Programação e sensores |

# Protótipos

## Trisistor P

Versão voltada para **introdução à eletrônica básica**.

Conteúdos:

- LEDs
- buzzers
- botões
- alimentação por pilha

📂 Pasta:

```
/trisistor-p
```

---

## Trisistor M

Versão voltada para **circuitos eletrônicos educativos**.

Exemplos:

- multivibrador astável
- sensor com LDR
- retardo de desligamento
- alarme com termistor
- circuito com diodo Zener
- circuito de histerese

📂 Pasta:

```
/trisistor-m
```

---

## Trisistor G

Versão programável com **Arduino e sensores**.

Exemplos de módulos:

- sensor ultrassônico
- sensor de toque
- servo motor
- sensor de luz
- buzzer

📂 Pasta:

```
/trisistor-g
```
