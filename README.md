# Trisistores

**Trisistores** é um projeto educacional de robótica e eletrônica desenvolvido para ensinar conceitos de tecnologia de forma progressiva.

O sistema é dividido em três níveis:

| Protótipo | Objetivo |
|--------|--------|
| Trisistor P | Introdução à eletrônica |
| Trisistor M | Circuitos eletrônicos |
| Trisistor G | Programação e sensores |

---

# Estrutura do Projeto

O repositório contém:

- documentação educacional
- esquemas eletrônicos
- arquivos 3D
- códigos Arduino
- material para professores e alunos

---

# Protótipos

## Trisistor P

Versão voltada para **introdução à eletrônica básica**.

Conteúdos:

- LEDs
- buzzers
- botões
- alimentação por pilha

📂 Pasta:

```
/trisistor-p
```

---

## Trisistor M

Versão voltada para **circuitos eletrônicos educativos**.

Exemplos:

- multivibrador astável
- sensor com LDR
- retardo de desligamento
- alarme com termistor
- circuito com diodo Zener
- circuito de histerese

📂 Pasta:

```
/trisistor-m
```

---

## Trisistor G

Versão programável com **Arduino e sensores**.

Exemplos de módulos:

- sensor ultrassônico
- sensor de toque
- servo motor
- sensor de luz
- buzzer

📂 Pasta:

```
/trisistor-g
```
